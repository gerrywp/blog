jsonp实际上就是服务器端实现对客户端函数的回调(callback)
<!--more-->
### 客户端浏览器

```js
<script type="text/javascript">
function test(data) {
alert(data);
}
</script>
<script type="text/javascript" src="http://******.aspx?callback=test"></script>
```

### 服务器端(c#-webform)

```cs
var fun = Request["callback"];
Response.Write(fun+"(\"你好世界\")");
```

> 其实就是这么个原理
