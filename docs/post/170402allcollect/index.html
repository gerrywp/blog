c++中const组合引用和指针

  1. const组合引用
    引用结合const,const关键字只有一个位置：指向常量的引用

    const int i=7;
const int& ci=i;

  2. const组合指针(因为指针是object类型) 指针结合const,const关键字有两个位置：(1)指向常量的指针 (2)指向常量的常指针

    //指向常量的指针
const int i=7;
const int* pci=&i;

//指向常量的常指针
const int* const cpci=&i;
